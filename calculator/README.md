# calculator

a simple command-line calculator built using python

---

## features

* **version1:** (python learnt till 11.04.26) supports basic operations: +, -, *, / and prevents division by zero to avoid ZeroDevisionError.

---

## how to run

run the file using:

```
python calculator.py
```
